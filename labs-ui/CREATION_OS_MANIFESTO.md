**(Swarm Intelligence)**

### Agent Roles
- **Orchestrator Agent**  
  Routes tasks to the optimal tool or model based on signal-to-noise, cost, and execution fidelity.

- **Connector Agent**  
  Builds glue code between tools using:
  - Official APIs  
  - Unofficial interfaces  
  - Headless browser automation  

- **Filter Agent**

Agents collaborate.  
The architect retains authority.
